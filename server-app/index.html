API DESCRIPTION

User Login API

Rejestracja

Path: /createUser POST METHOD

Params:

email

password

Sample response if correct:

{ "status": true, "message": "User created" }

Sample response if incorrect:

{ "status": false, "message": "User already exists" }

Token Login

Path: /tokenLogin POST METHOD

Headers:

token

Sample response if correct:

{ "status": true, "message": "Valid token. Logged" }

Sample response if incorrect:

{ "status": false, "message": "Token invalid" }
